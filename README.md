# POS System

A desktop Point of Sale application built with Python and Tkinter, backed by SQLite.

## Features

- Multi-role authentication (Admin, Manager, Cashier)
- Product and inventory management
- Sales processing with cash, MoMo, and card payments
- Reports and analytics
- Customer loyalty tracking
- Automatic database migrations

## Requirements

- Python 3.11+
- Windows 10/11

## Setup

1. Clone the repository
2. Create a virtual environment
